# NeoComerZ UI প্যাকেজ ম্যানেজমেন্ট এবং ওয়ার্কফ্লো ডকুমেন্টেশন

## সারসংক্ষেপ

এই ডকুমেন্টে NeoComerZ Storefront UI প্যাকেজের সম্পূর্ণ প্যাকেজ ম্যানেজমেন্ট সিস্টেম এবং কার্যপ্রবাহ বাংলায় বর্ণনা করা হয়েছে।

---

## ১. প্যাকেজের পরিচিতি

### প্যাকেজ তথ্য
- **নাম**: `neocomerz-storefront-ui`
- **ভার্সন**: `1.0.6`
- **লাইসেন্স**: MIT
- **লেখক**: NeoComerZ
- **রিপোজিটরি**: https://github.com/neocomerz/neocomerz-ui

### বর্ণনা
NeoComerZ স্টোরফ্রন্টের জন্য প্রিমিয়াম UI কম্পোনেন্ট লাইব্রেরি যা React এবং Next.js এর জন্য অপ্টিমাইজ করা।

---

## ২. প্যাকেজ স্ট্রাকচার

```
neocomerz-ui/
├── src/                    # সোর্স কোড
│   ├── components/         # সব কম্পোনেন্ট
│   │   ├── Button/
│   │   ├── Hero/
│   │   ├── Gallery/
│   │   ├── OrderForm/
│   │   └── ... (আরও অনেক)
│   ├── types/              # TypeScript টাইপ ডেফিনেশন
│   ├── images/             # ইমেজ ফাইল
│   └── index.ts           # মূল এক্সপোর্ট ফাইল
├── preview/               # ডেভেলপমেন্ট প্রিভিউ
├── dist/                  # বিল্ড আউটপুট (জেনারেটেড)
├── package.json           # প্যাকেজ কনফিগারেশন
├── package-lock.json      # ডিপেন্ডেন্সি লক ফাইল
├── tsconfig.json          # TypeScript কনফিগ
├── tailwind.config.ts     # Tailwind CSS কনফিগ
└── postcss.config.mjs     # PostCSS কনফিগ
```

## ৬. কম্পোনেন্ট আর্কিটেকচার

### কম্পোনেন্ট ক্যাটাগরি

#### **হিরো সেকশন**
- `Hero` - ক্লাসিক লার্জ হিরো
- `LayeredHero` - মাল্টিলেয়ার হিরো
- `InlineHero` - মিনিমাল ইনলাইন হিরো

#### **গ্যালারি**
- `GalleryGrid` - ফিচার গ্রিড
- `GalleryGrid6` - ৬ আইটেম গ্রিড
- `GalleryCol` - কলাম গ্যালারি

#### **কমার্স**
- `OrderForm` - অর্ডার ফর্ম
- `SizeChart` - সাইজ চার্ট

#### **ফিডব্যাক ও ইনফো**
- `Testimonials` - কাস্টমার টেস্টিমনিয়াল
- `FAQ` - এফএকিউ সেকশন
- `Quality` - কোয়ালিটি সেকশন
- `Helpline` - হেল্পলাইন বাটন

#### **ইউটিলিটি**
- `PrimaryButton` - প্রাইমারি বাটন
- `SectionHeader` - সেকশন হেডার
- `CheckListItem` - চেকলিস্ট আইটেম

### ভার্সনিং সিস্টেম
প্রতিটি কম্পোনেন্টের একাধিক ভার্সন আছে:
- `ComponentName` (ডিফল্ট)
- `ComponentNameV0` থেকে `ComponentNameV5`

## ৭. ওয়ার্কফ্লো প্রক্রিয়া

### স্টেপ ১: ডেভেলপমেন্ট সেটআপ
```bash
# রিপোজিটরি ক্লোন
git clone https://github.com/neocomerz/neocomerz-ui.git

# ডিপেন্ডেন্সি ইনস্টল
npm install

# ডেভেলপমেন্ট সার্ভার চালু
npm run dev
```

### স্টেপ ২: কম্পোনেন্ট ডেভেলপমেন্ট
1. `src/components/` ফোল্ডারে নতুন কম্পোনেন্ট তৈরি
2. প্রতিটি কম্পোনেন্টের জন্য ফোল্ডার স্ট্রাকচার:
   ```
   ComponentName/
   ├── index.tsx         # মূল কম্পোনেন্ট
   ├── ComponentNameV0.tsx
   ├── ComponentNameV1.tsx
   └── ... (অন্যান্য ভার্সন)
   ```

### স্টেপ ৩: টাইপ ডেফিনেশন
- `src/types/` ফোল্ডারে TypeScript টাইপ ডেফাইন
- প্রপস ইন্টারফেস তৈরি করুন

### স্টেপ ৪: এক্সপোর্ট ম্যানেজমেন্ট
- `src/index.ts` ফাইলে সব কম্পোনেন্ট এক্সপোর্ট করুন
- টাইপ ডেফিনেশনও এক্সপোর্ট করুন

### স্টেপ ৫: বিল্ড প্রক্রিয়া
```bash
# লাইব্রেরি বিল্ড
npm run build

# আউটপুট ফাইল
dist/
├── index.js      # CommonJS
├── index.mjs     # ES Module
└── index.d.ts    # TypeScript টাইপ
```

## ৯. ইনস্টলেশন ও ব্যবহার


```tsx
import { Hero, PrimaryButton } from 'neocomerz-storefront-ui';

export default function Home() {
  return (
    <main>
      <Hero 
        title="আপনার স্টাইল উন্নত করুন"
        subtitle="এক্সক্লুসিভ সামার কালেকশন"
        primaryColor="#F36621"
      />
      <PrimaryButton href="/shop">এখনই কিনুন</PrimaryButton>
    </main>
  );
}
```
